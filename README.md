# 🏍️ Бот-щоденник обслуговування для мотоцикла

Telegram-бот для відстеження обслуговування мотоцикла з нагадуваннями та красивим інтерфейсом кнопок.

## ✨ Функціональність

- 📊 **Відстеження пробігу** - записуйте поточний пробіг мотоцикла
- 🔧 **Обслуговування** - ведіть записи про заміну масла, фільтрів, регулювання клапанів
- 🔗 **Змащення ланцюга** - нагадування кожні 500 км
- � **Статистика** - переглядайте статистику обслуговування та витрат
- ⚙️ **Налаштування** - встановлюйте інтервали для нагадувань
- � **Історія** - переглядайте всю історію обслуговування
- 📤 **Експорт** - експортуйте дані в CSV
- ⏰ **Нагадування** - автоматичні нагадування про обслуговування
- 🎨 **Красивий інтерфейс** - зручні кнопки для навігації

## 📋 Команди бота

| Команда | Опис |
|---------|------|
| `/start` | Початок роботи з ботом (головне меню) |
| `/help` | Довідка по командах |
| `/chain` | Швидке записування змащення ланцюга |

**Використовуйте кнопки меню для всіх інших функцій!**

## 🚀 Встановлення

### 1. Клонування або створення проекту

Переконайтеся, що ви в директорії проекту:
```bash
cd c:\Users\user\Desktop\python\motorcycle_diary_bot
```

### 2. Створення віртуального середовища

```bash
python -m venv venv
```

### 3. Активація віртуального середовища

**Windows:**
```bash
venv\Scripts\activate
```

### 4. Встановлення залежностей

```bash
pip install -r requirements.txt
```

### 5. Отримання токену бота

1. Знайдіть @BotFather в Telegram
2. Надішліть `/newbot` та слідуйте інструкціям
3. Скопіюйте отриманий токен

### 6. Налаштування змінних середовища

Створіть файл `.env` в корені проекту:
```bash
TELEGRAM_BOT_TOKEN=ваш_токен_тут
```

Або скопіюйте приклад:
```bash
copy .env.example .env
```

І відредагуйте `.env`, додавши ваш токен.

## ▶️ Запуск бота

```bash
python bot.py
```

Бот запуститься і буде готовий до роботи!

## 📝 Використання

### Головне меню

Після запуску бота натисніть `/start` для відображення головного меню з кнопками:

- 📊 **Пробіг** - оновити поточний пробіг
- 🔧 **Обслуговування** - додати запис про обслуговування
-  **Статистика** - перегляд статистики та експорт даних
- ⚙️ **Налаштування** - налаштувати інтервали для нагадувань
- 📋 **Історія** - перегляд історії обслуговування
- 🔗 **Ланцюг** - швидке записування змащення ланцюга

### Оновлення пробігу

1. Натисніть кнопку "📊 Пробіг"
2. Введіть поточний пробіг в км
3. Бот автоматично перевірить, чи потрібно змастити ланцюг

### Додавання запису про обслуговування

1. Натисніть кнопку "🔧 Обслуговування"
2. Оберіть тип обслуговування з меню (масло, фільтри, клапани, інше)
3. Введіть пробіг на момент обслуговування
4. Додайте нотатку (необов'язково, або /skip щоб пропустити)
### Змащення ланцюга

1. Натисніть кнопку "🔗 Ланцюг"
2. Бот автоматично запише поточний пробіг як момент змащення
3. Наступне нагадування через 500 км

### Налаштування інтервалів

1. Натисніть кнопку "⚙️ Налаштування"
2. Оберіть тип обслуговування (масло, фільтри, клапани)
3. Введіть інтервал в км
4. Бот буде нагадувати про обслуговування згідно з налаштуваннями

### Статистика та експорт

1. Натисніть кнопку "📈 Статистика"
2. Перегляньте статистику обслуговування та витрат
3. Натисніть "📤 Експорт в CSV" для отримання даних

## ⏰ Автоматичні нагадування

Бот автоматично відправляє нагадування:

- **Щотижня:** "Привіт! Не забудь оновити поточний пробіг твого мотоцикла"
- **Кожні 6 годин:** перевіряє, чи проїхали ви 500 км після останнього змащення ланцюга

## 🗄️ База даних

Бот використовує SQLite базу даних (`motorcycle_diary.db`), яка створюється автоматично при першому запуску.

Структура бази:
- `users` - інформація про користувачів та поточний пробіг
- `maintenance_records` - записи про обслуговування
- `maintenance_intervals` - налаштування інтервалів обслуговування
- `reminders` - інформація про нагадування

## 🛠️ Технології

- python-telegram-bot 20.7
- python-dotenv 1.0.0
- SQLite3

## 📝 Приклади використання

Бот простий у використанні - просто натискайте кнопки меню та слідуйте інструкціям.

1. `/start` → натисніть "🔧 Обслуговування"
2. Оберіть "🛢️ Заміна масла"
3. Введіть пробіг: `5000`

5. Готово! Запис збережено.

### Приклад налаштування нагадувань:

1. `/start` → натисніть "⚙️ Налаштування"
2. Оберіть "🛢️ Масло"
3. Введіть інтервал: `5000`
4. Тепер бот нагадає про заміну масла кожні 5000 км.

## 🔒 Безпека

- Токен бота зберігається в `.env` файлі (не додавайте його в git)
- База даних створюється локально
- Кожен користувач має доступ тільки до своїх даних

## 🐛 Вирішення проблем

**Бот не запускається:**
- Перевірте, чи встановлені всі залежності: `pip install -r requirements.txt`
- Перевірте, чи правильний токен в `.env` файлі

**Нагадування не приходять:**
- Переконайтеся, що бот запущений
- Перевірте, чи бот має доступ до відправки повідомлень
